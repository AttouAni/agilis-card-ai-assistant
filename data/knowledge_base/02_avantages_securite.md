# Sécurité de la carte AGILIS

AGILIS permet d'effectuer les transactions en toute sécurité grâce aux dispositifs suivants :

1. **Puce électronique de dernière génération** : AGILIS est équipée d'une puce assurant une utilisation simple combinée à une sécurité maximale.

2. **Code confidentiel obligatoire** : un code confidentiel (PIN) est attribué à chaque utilisateur. Aucune transaction ne peut être validée sans la saisie de ce code. À chaque transaction, un ticket détaillé est remis, mentionnant :
   - la date et l'heure,
   - la station-service,
   - le numéro de la carte,
   - le solde avant et après la transaction,
   - le produit prélevé,
   - le montant et la quantité prélevés,
   - le kilométrage (si l'option est activée).

3. **Codes multiples possibles** : il est possible d'associer à une même carte un code PIN principal et plusieurs codes chauffeurs sécurisés, chaque chauffeur ayant son propre code unique.

4. **Blocage automatique après 3 essais erronés** : la carte AGILIS est bloquée si un code erroné est saisi trois fois consécutivement. Toute carte mise en opposition est automatiquement refusée par le TPE (Terminal de Paiement Électronique).

5. **TPE intégrés aux distributeurs** : Agil Energy dispose d'une nouvelle génération de distributeurs automatiques de carburant intégrant directement les Terminaux de Paiement Électronique, asservissant la vente du produit à la transaction. Cela renforce la sécurité et la transparence des transactions en station.

6. **Transactions en ligne (online), avec mode dégradé limité** : toutes les transactions sont effectuées en ligne (online). En cas de défaillance du réseau reliant le TPE au serveur, un nombre limité de transactions "offline" est autorisé par carte et par station.

7. **Chiffrement de bout en bout** : les transactions sont cryptées de bout en bout.

8. **Traçabilité détaillée** : toutes les transactions sont détaillées par véhicule, par utilisateur et par point de vente. Le suivi de ces opérations d'approvisionnement en carburant peut être consulté quotidiennement via le portail web AGILIS ONLINE.
